Aktivasi Online

Mengaktivasi Software TigaPutri secara online

Untuk mengaktivasi Software TigaPutri secara online, langkah-langkahnya sebagai berikut:

  1. Pastikan sudah menginstall Software TigaPutri-nya sampai bisa dijalankan. Untuk cara instalasinya bisa dibaca di sini

  2. Pastikan Edisi Software TigaPutrinya sudah sesuai. Jika TigaPutri-nya masih Free Edition, silakan hubungi pengembang TigaPutri untuk mendapatkan file TigaPutri sesuai edisi yang mau diaktivasi

  3. Jalankan Software TigaPutri-nya, kemudian klik menu Lain-Lain → Aktivasi

  4. Jika sudah membuat AppID langsung ke langkah. Jika belum, pada bagian 1. Buat AppID anda isikan Nama Server dan Nama Pemilik kemudian klik Generate App ID untuk membuat AppID

caution

Pastikan Nama Server dan Nama Pemilik sudah benar, karena ketika TigaPutri sudah diaktivasi, maka perubahan pada Nama Server dan Nama Pemilik memerlukan aktivasi ulang, dan mungkin akan dikenakan biaya untuk perubahan aktivasi

  1. Lanjut di bagian 2. Migrasi \/ Pembelian \/ Pembayaran Iuran, pilih Beli Lisensi kemudian klik OK. Maka akan muncul formulir Pembelian Lisensi TigaPutri

  2. Silakan isi data anda di Pembelian Lisensi TigaPutri

  3. Bila sudah, klik Cek Harga disebelah kanan untuk cek harga pembelian lisensi baru TigaPutri

  4. Setelah harganya muncul, pilih Metode Pembayaran (untuk saat ini, metode pembayaran yang tersedia hanya Transfer Bank). Kemudian klik Beli untuk mengirim request pembelian

  5. Setelah itu, akan tampil jumlah tagihan yang harus dibayar. Silakan transfer sesuai nominal yang tertera sebelum batas waktu pembayaran berakhir. Pastikan nominal yang ditransfer sesuai nominal yang ditampilkan agar bisa terverifikasi otomatis

  6. Jika sudah mentransfer, silakan klik Cek Status Pembayaran. Jika pembayaran telah diverifikasi, maka akan muncul konfirmasi untuk mengaktivasi TigaPutri, klik OK

  7. TigaPutri akan otomatis teraktivasi

  8. Setelah itu, tutup dan buka lagi TigaPutrinya
